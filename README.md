###  Featured Projects

Here are some of the key projects I have built, ranging from AI applications to custom embedded hardware:

####  AI & Intelligent Automation

*   **[Kchiro](https://github.com/Madhura-kene/Kchiro)** 
    *A locally-run 3D asset generation tool built on Codex and powered by Ollama. It produces high-quality assets for games, films, and architectural visualization with built-in export support.*
    * **Tech Stack:** Python, Ollama, AI Generation, 3D Assets

*   **[Assignment Likhne Vala](https://github.com/Madhura-kene/assignment-likhne-vala)**  
    *Stop struggling with margins! This tool perfectly aligns digital text to physical assignment sheets. Includes multi-page PDF support and realistic messy handwriting fonts.*
    * **Tech Stack:** JavaScript, HTML/CSS, PDF Generation

####  Hardware & IoT Systems

*   **[ESP32 Air Mouse](https://github.com/Madhura-kene/ESP32-controler)**   
    *A Bluetooth Low Energy (BLE) HID Air Mouse built with an ESP32, an MPU-6050 IMU for motion tracking, and a TTP223 capacitive touch sensor for cursor movement and clicking.*
    * **Tech Stack:** C++, ESP32, MPU-6050, BLE HID

*   **[NullBeam Laser Alarm](https://github.com/Madhura-kene/NullBeam)**   
    *A compact, ESP32-based security system using a continuous laser beam and an LDR sensor. When the beam is broken, the alarm latches (buzzer stays on) and sends an instant Telegram alert.*
    * **Tech Stack:** C++, ESP32, Telegram Bot API, Hardware Integration

*   **[RakeRadar Passenger Crowd Mapping](https://github.com/Madhura-kene/rakeradar)**   
    *Real-time passenger crowd mapping and RakeVision AI compartment occupancy scanner for the Pune-Lonavala suburban transit network.*
    * **Tech Stack:** JavaScript, Node.js, AI Computer Vision, Full-Stack

*   **[Medicine Reminder Box](https://github.com/Madhura-kene/medicine-reminder)**   
    *An Arduino Uno based medicine reminder that triggers a 5-LED chaser animation and piezo buzzer simultaneously for 10 seconds at scheduled times (1:30 PM and 8:30 PM daily).*
    * **Tech Stack:** C++, Arduino Uno, Hardware Control

*   **[Desk Bot](https://github.com/Madhura-kene/desk-bot)**  
    *An ESP32 OLED face controller for interactive desk companion robots.*
    * **Tech Stack:** C++, ESP32, OLED Display
