### Featured Projects

#### AI & Intelligent Automation

* **[Kchiro](https://github.com/Madhura-kene/Kchiro)**
  * A locally-run 3D asset generation tool built on Codex and powered by Ollama. It produces high-quality assets for games, films, and architectural visualization with built-in export support.
  * **Tech Stack:** Python, Ollama, AI Generation, 3D Assets

* **[Assignment Likhne Vala](https://github.com/Madhura-kene/assignment-likhne-vala)**
  * A digital-to-analog layout script designed to align digital text correctly onto scanned physical sheets. Includes multi-page PDF generation and realistic handwriting font overlays.
  * **Tech Stack:** JavaScript, HTML/CSS, PDF Generation

#### Hardware & IoT Systems

* **[ESP32 Air Mouse](https://github.com/Madhura-kene/ESP32-controler)**
  * A Bluetooth Low Energy (BLE) HID Air Mouse built with an ESP32, utilising an MPU-6050 IMU for motion tracking and a TTP223 capacitive touch sensor for click inputs.
  * **Tech Stack:** C++, ESP32, MPU-6050, BLE HID

* **[NullBeam Laser Alarm](https://github.com/Madhura-kene/NullBeam)**
  * A compact security system using a continuous laser beam and LDR receiver. Features an alarm latch state and instant notifications via the Telegram Bot API.
  * **Tech Stack:** C++, ESP32, Telegram Bot API, Hardware Integration

* **[RakeRadar Passenger Crowd Mapping](https://github.com/Madhura-kene/rakeradar)**
  * Real-time compartment occupancy scanning and passenger density mapping system for the Pune-Lonavala suburban transit network.
  * **Tech Stack:** JavaScript, Node.js, AI Computer Vision, Full-Stack

* **[Medicine Reminder Box](https://github.com/Madhura-kene/medicine-reminder)**
  * An automated schedule reminder that manages sequential LED indicators and piezo alarm signals to help track daily prescription times.
  * **Tech Stack:** C++, Arduino Uno, Hardware Control

* **[Desk Bot](https://github.com/Madhura-kene/desk-bot)**
  * A custom OLED face state controller for interactive desktop companion robots.
  * **Tech Stack:** C++, ESP32, OLED Display
